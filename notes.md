  Prints a list of 5(?) events in Memphis
  Gives the event name and time of event
  Asks user which event to learn more about

    Can also type exit to quit the program
  Goes to detail page of the event

  Event name
  Event date/time
  Event details url

Details:
  Event name
  Event date/time
  Event location
  Event summary
  Link to FB event
